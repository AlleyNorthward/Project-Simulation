# 常用函数
## QApplication
- 首先, 这个构造函数, 需要调用`argc`以及`argv`. 那么, `argc`, `argv`是什么呢? 有什么用处呢? 
    - 这俩参数是`int main`中传入的. 实际上, 这其实就是我们在命令行端调用的传入的参数. 其中, `argc`是参数数量, `argv`是参数具体内容. 确实, 通过这俩参数, 就能准确获取到我们命令行端具体的参数. 
    - 但是, 对于`Qt`这个构造函数为什么要调用呢? 我也不知道, 哈哈. 可能是为了更精确地控制吧. 
    - 我们知道这些参数有什么作用就行. 
- 好像知道传入参数的作用就行了吧, 具体实现肯定说不清道不明的, 就这样吧. 
## exec
- 这个东西呢, 其实就跟嵌入式中`main`函数的`while`循环一样, 一直执行着事件循环. 而且是必须的, 否则的话, `main`函数直接终止了, 也就无从谈起`Qt`的生命周期了.
